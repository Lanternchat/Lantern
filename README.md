# Lantern — Demo local

Archivos:
- `index.html`
- `styles.css`
- `script.js`

Instrucciones rápidas:
1. Guarda los archivos en la misma carpeta.
2. Desde esa carpeta inicia un servidor local.

Comandos (PowerShell / Terminal):
```bash
# Node (npx)
npx --yes http-server . -p 8080

# Python 3
python -m http.server 8080
